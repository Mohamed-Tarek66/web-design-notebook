The css code is written in the following form:

selector,
selector,
selector
{
  property : value;
  property : value;
  property : value;
  property : value
}

This following list contains some commom selectors:

selector usage
tag it will apply the properties for every tag of that type inside the web page
.class or tag.class it will apply the properties for every tag of that class inside the web page
#id it will apply the properties for the tag of that id inside the web page
tag:pseudo class it will apply the properties for every tag of that type that defined by that pseudo class the inside the web page
* it will apply the properties for all tags inside the web page
body it will apply the properties for the web page itself
tag1 tag2 it will apply the properties for every tag of the type tag2 inside a tag of that type tag1
tag1>tag2 it will apply the properties for every tag of the type tag2 directly inside a tag of that type tag1
tag1~tag2 it will apply the properties for every tag of the type tag2 sharing the same father with a tag of that type tag1
tag1+tag2 it will apply the properties for every tag of the type tag2 after a tag of that type tag1 and they share the same father
[attribute] it will apply the properties for all tags that have that attribute
tag[attribute] it will apply the properties for all tags of that tag type that have that attribute
[attribute="value"] it will apply the properties for all tags that have this exact value for that attribute
[attribute^="value"] it will apply the properties for all tags that start with this exact value for that attribute
[attribute$="value"] it will apply the properties for all tags that end with this exact value for that attribute
[attribute*="value"] it will apply the properties for all tags that contain this exact value for that attribute
tag[attribute="value"] it will apply the properties for all tags of that tag type that have this exact value for that attribute
::selection it will apply the properties for any selected text
tag::selection it will apply the properties for any selected text inside a tag of that tag type
tag::after used to add and apply certain properties for the added contents after any tag of that type
tag::before used to add and apply certain properties for the added contents before any tag of that type
tag::first-letter it will apply the properties for the first litter of text inside a tag of that tag type
tag::first-line it will apply the properties for the first line of text inside a tag of that tag type
:not(selector) it will apply the properties for any tag that does not achieve the criteria of that selector
tag:not(selector) it will apply the properties for any tag of that tag type that does not achieve the criteria of that selector

Common pseudo classes are mentioned in the following table:

pseudo class usage
only-child the properties will be applied to any tag in case that it is the only tag inside a tag of the same type as the tag mentioned in the selector
first-child the properties will be applied to any tag in case that it is the first tag inside a tag of the same type as the tag mentioned in the selector
last-child the properties will be applied to any tag in case that it is the last tag inside a tag of the same type as the tag mentioned in the selector
nth-child(rule) the properties will be applied to any tag in case that it achieves the rule of being the nth tag inside a tag of the same type as the tag mentioned in the selector
the rule could be:
  • a number
  • even
  • odd
  • equation with variable n
nth-last-child(rule) the properties will be applied to any tag in case that it achieves the rule of being the nth tag counting from the end inside a tag of the same type as the tag mentioned in the selector
the rule could be:
  • a number
  • even
  • odd
  • equation with variable n
only-of-type the properties will be applied to any tag in case that it is the only tag of its own type inside a tag of the same type as the tag mentioned in the selector
first-of-type the properties will be applied to any tag in case that it is the first tag of its own type inside a tag of the same type as the tag mentioned in the selector
last-of-type the properties will be applied to any tag in case that it is the last tag of its own type inside a tag of the same type as the tag mentioned in the selector
nth-of-type(rule) the properties will be applied to any tag in case that it achieves the rule of being the nth tag of its own type inside a tag of the same type as the tag mentioned in the selector
the rule could be:
  • a number
  • even
  • odd
  • equation with variable n
nth-last-of-type(rule) the properties will be applied to any tag in case that it achieves the rule of being the nth tag of its own type counting from the end inside a tag of the same type as the tag mentioned in the selector
the rule could be:
  • a number
  • even
  • odd
  • equation with variable n
enabled the properties will be applied to any enabled tag with the same tag type as the tag mentioned in the selector
disabled the properties will be applied to any disabled tag with the same tag type as the tag mentioned in the selector
empty the properties will be applied to any tag with the same tag type as the tag mentioned in the selector that does not contain any text or inner tags
checked the properties will be applied to any checked tag with the same tag type (checkbox input, radio input) as the tag mentioned in the selector
hover the properties will be applied to any tag with the same tag type as the tag mentioned in the selector when the user stands with the mouse over it
active the properties will be applied to any tag with the same tag type as the tag mentioned in the selector as long as the focuse is on it
root the properties will be applied to the external tag of the code meaning the HTML tag

The selector can be a chain of selectors as following:

  • tag .class .......
  • .class .class ......
  • .class tag ......

The comment in CSS is written in the following form /* the comment */.
