The transform property is used to change the position and form of the tag from its original position and form in several ways as mentioned below:

  • Scale the tag

    • Used to scale the tag in order to make it bigger or smaller than its original size. Its value can be in one of the following forms:

    form indication
    scale(no) the dimentions of the tag will be its original dimentions multiplied by that number
    scale(no, no) the X dimention of the tag will be its original X dimention multiplied by the first number, while the Y dimention of the tag will be its original Y dimention multiplied by the second number
    scalex(no) the X dimention of the tag will be its original X dimention multiplied by that number
    scaley(no) the Y dimention of the tag will be its original Y dimention multiplied by that number
    scalez(no) the tag will be scaled in the Z direction based on that number (the tag must be scaled in both or either the X or Y direction beside the Z direction)
    scale3d(no, no, no) the X dimention of the tag will be its original X dimention multiplied by the first number, while the Y dimention of the tag will be its original Y dimention multiplied by the second number and the last number scales the tag in the Z direction
  • Rotate the tag

    • Used to rotate the tag around the X, Y or Z axis. Its value can be in one of the following forms:

    form indication
    rotate(nodeg) rotates the tag around the Z axis by this degree
    rotate(nodeg, nodeg, nodeg) rotates the tag around the X axis by the first degree, around the Y axis by the second degree & around the Z axis by the third degree
    rotatex(nodeg) rotates the tag around the X axis by this degree
    rotatey(nodeg) rotates the tag around the Y axis by this degree
    rotatez(nodeg) rotates the tag around the Z axis by this degree
  • Translate the tag

    • Used to move the tag from its original position. Its value can be in one of the following forms:

    form indication
    translate(nopx, nopx) moves the tag horizontally with the first number & vertically with the second number
    translate3d(nopx, nopx, nopx) moves the tag horizontally with the first number, vertically with the second number & towards the user with the third number
    translatex(nopx) moves the tag horizontally with that number
    translatey(nopx) moves the tag vertically with that number
    translatez(nopx) moves the tag towards the user with that number (if the tag was translated with a number larger than the perspective it will disappear as it moved pass the user)
  • Skew the tag

    • Used to move the border of the tag in opposite direction. Its value can be in one of the following forms:

    form indication
    skew(nodeg, nodeg) skews the tag horrizontally with the first degree & vertically with the second degree
    skewx(nodeg) skews the tag horrizontally with this degree
    skewy(nodeg) skews the tag vertically with this degree

In order to collect all the transform values one of the following forms are used:

selector{
  transform: value(...) value(...) value(...) value(...) value(...)
}
selector{
  transform: matrix(no of scalex, deg of skewy, deg of skewx, no of scaley, px of translatex, px of translatey)
}
selector{
  transform: matrix3d(values from matrix3d tool)
}

The transform property determines the form the tags transform with. Its value can be one of the following:

value indication
flat the inner tags are stacked onto the transforming tag moving with its movement
preserve-3d the inner tags are not stacked onto the transforming tag so every tag can move on itown

The transition-property property determines the css animated properties that would be affected bt the tags transform. Its value can be one of the following:

value indication
all all css animated properties will be affected bt the tags transform
cssAnimatedProperty only this css animated property will be affected bt the tags transform
cssAnimatedProperty, cssAnimatedProperty, cssAnimatedProperty, .... these css animated properties will be affected bt the tags transform

The transition-duration property determines the time that the tag takes to transform from its original position. Its value can be in one of the following form:

form indication
nos the time in secomds
noms the time in millisecomds

The transition-delay property determines the time that the tag wait for before transforming from its original position. Its value can be in one of the following form:

form indication
nos the time in secomds
noms the time in millisecomds

The transition-timing-function property determines the spead form at which the tag transforms from its original position. Its value can be one of the following:

tag{transition-timing-function: ease}

tag{transition-timing-function: linear}

tag{transition-timing-function: ease-in}

tag{transition-timing-function: ease-out}

tag{transition-timing-function: ease-in-out}

tag{transition-timing-function: strps(5, start)}

tag{transition-timing-function: strps(5, end)}
