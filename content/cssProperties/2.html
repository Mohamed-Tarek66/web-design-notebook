The background-color property is used to determine the color of the background of the selector. Its value can be in one of the following forms:

value form example
the name of the color red
the colour codes #cecf93
the redgreenblue function rgb(101, 150, 156)
the redgreenblueopacity function rgba(117, 138, 204, 0.68)

The background-image property is used to determine the source of the background image of the selector. Its value is written in this form url(the location of the image).

The background-repeat property is used to determine whether the background image of the selector will be repeated or not. Its value can be one of the following:

value indication
no-repeat the image will not be repeated neither in the X nor Y directions
repeat-x the image will only be repeated in the X direction to cover the selector horizontally
repeat-y the image will only be repeated in the Y direction to cover the selector vertically
repeat the image will be repeated in both the X & Y directions to cover the intire selector

The background-attachment property is used to determine whether the background image of the selector will move with scrolling or will be fixed in its position. Its value can be one of the following:

value indication
scroll the image will move with scrolling and will not be fixed in its position
fixed the image will be fixed in its position as the user scroll

An example about the background-attachment property:

tag{
background-attachment: scroll
}
scroll the page
tag{
background-attachment: fixed
}
scroll the page

The background-size property is used to determine the size that the background image of the selector will be set to. Its value can be in one of the following forms:

value form indication
auto auto the image will stick with its original size value
no% no% the image size will be a percentage of the tag size
nopx nopx the image size will be determined acording to the pixel value
contain the image will change its size so that either its hight or its width will be equal to the hight or width of the tag
cover the image will upscale its size so that its width will be equal to the width of the tag this makes sure that the image is covering the intire tag all the time

An example about the background-size property:

tag{
background-size: auto auto;
background-repeat: no-repeat
}
tag{
background-size: 50% 50%;
background-repeat: no-repeat
}
tag{
background-size: 100px 100px;
background-repeat: no-repeat
}
tag{
background-size: contain;
background-repeat: no-repeat
}
tag{
background-size: cover;
background-repeat: no-repeat
}

The background-origin property determines the start point of the background image.The background-clip property is similar but for the background color. their value can be one of the following:

value indication
border-box the background and the border of the tag will share the same start point
padding-box the background and the padding will share the same start point
content-box the background and the content of the tag will share the same start point

An example about the background-origin property & the background-clip property:

tag{
background-origin: border-box;
background-size: 100% 100%;
background-repeat: no-repeat
}
tag{
background-origin: padding-box;
background-size: 100% 100%;
background-repeat: no-repeat
}
tag{
background-origin: content-box;
background-size: 100% 100%;
background-repeat: no-repeat
}
tag{
background-clip: border-box;
background-size: 100% 100%;
background-repeat: no-repeat
}
tag{
background-clip: padding-box;
background-size: 100% 100%;
background-repeat: no-repeat
}
tag{
background-clip: content-box;
background-size: 100% 100%;
background-repeat: no-repeat
}
