The border-width property is used to determine the width of the border of the selector. Its value can be in one of the following forms:

value form idication
nopx indicates the width of the top, bottom, right and left border
nopx nopx the first no indicates the width of the top and bottom border and the second no indicates the width of the right and left border
nopx nopx nopx the first no indicates width of the top border where the second no indicates the width of the right and left border and the third no indicates the width of the bottom border
nopx nopx nopx nopx the first, second, third, fourth no indicates the top, right, bottom, left borders by order
no% indicates the width of the top, bottom, right and left border
no% no% the first no indicates the width of the top and bottom border and the second no indicates the width of the right and left border
no% no% no% the first no indicates width of the top border where the second no indicates the width of the right and left border and the third no indicates the width of the bottom border
no% no% no% no% the first, second, third, fourth no indicates the top, right, bottom, left borders by order

The border-width property can be used with only one side of the selector or with each side alone forming the next four properties:

  • border-width-top
  • border-width-right
  • border-width-bottom
  • border-width-left

The border-style property is used to determine the shape of the border of the selector. Its value can be one of the following:

  • solid
  • dashed
  • dotted
  • double

The border-style property can be used with only one side of the selector or with each side alone forming the next four properties:

  • border-style-top
  • border-style-right
  • border-style-bottom
  • border-style-left

The border-color property is used to determine the color of the border of the selector. Its value can be in one of the following forms:

value form example
the name of the color red
the colour codes #cecf93
the redgreenblue function rgb(101, 150, 156)
the redgreenblueopacity function rgba(117, 138, 204, 0.68)

The border-color property can be used with only one side of the selector or with each side alone forming the next four properties:

  • border-color-top
  • border-color-right
  • border-color-bottom
  • border-color-left

The border property is used to collect some or all the information about the border of the tag in one property. it is witten in the following form:

selector
{
  border : border-width border-style border-color
}

The border property can be used with only one side of the selector or with each side alone forming the next four properties:

  • border-top
  • border-right
  • border-bottom
  • border-left

The outline property is similar to the border property except that it is not part of the selector's size. the following example showes the difference:

tag{
border: 10px solid red;
outline: 10px solid green
}
