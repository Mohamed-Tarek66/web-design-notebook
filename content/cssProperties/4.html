The direction property is used to determine direction of the text. Its value can be one of the following:

value indication
ltr from left to right
rtl from right to left

The font-family property is used to determine shape of the text font. Its value can be in one of the following forms:

value indication
font name either a web safe font that is identified by most browsers (web safe fonts) like Arial or identified in the following form:
@font-face{
   font-family : font name;
   src : url('location of the font identification file')
}
"font name" a font name that consists of multiple words
font name 1, font name 2, font name 3, ..... multiple font names in case that the browser does not identified the first one

The white-space property is used to determine whether the text will wrap to the next line or not. Its value can be in one of the following forms:

tag{
  white-space : normal
}

the text will continue in the next line when it meets a space or special character or if it reatches the widths

tag{
  white-space : nowrap
}

the text will not continue in the next line but it will stay in the same line outside the tag or apper through scroll

The word-wrap property is used to determine whether the text will break words to wrap to the next line or not. Its value can be in one of the following forms:

tag{
  white-space : normal;
  word-wrap : normal
}

the woed will continue in the same line if it is longer than the tag width (decreaseTheWidthOfYourWindow)

tag{
  white-space : normal;
  word-wrap : break-word
}

the word will continue in the next line when it reatches the widths breaking (decreaseTheWidthOfYourWindow)

The text-overflow property is used to repalce the text outside the tag by "...". Its value can be in one of the following:

value indication
clip overflow text will continue out of the tag when it gets bigger than the tag
ellipsis overflow text will be replaced by "..." when it gets bigger than the tag

The text-transform property is used to make the text capital or small. Its value can be in one of the following:

tag{
  text-transform: capitalize;
}
the first litter of every word will be capital
tag{
  text-transform: uppercase;
}
all litters will be capital
tag{
  text-transform: lowercase;
}
all litters will be small
