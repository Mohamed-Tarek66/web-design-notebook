The column-count property separates a block of text into several columns like a journal. Its value can be in the following form: tag{column-count: no}

The column-width property determines the width of the columns. Its value can be in one of the following form:

form indication
auto the available width will be devided by the number of columns
nopx this value will be the width of each column

The columns property is used to collect the values of the column-count & column-width properties together. it is witten in the following form:

selector{
  columns : column-width column-count
}

The column-span property determines whether the outline of the article part is of the first column or not. Its value can be one of the following:

value indication
1 the outline of the article is part of the first column
all the outline of the article is above the entire article

The column-gap property determines the width of the space between each 2 columns. Its value can be in the following form: tag{column-gap: nopx}

The column-rule-width property determines the width of the borderline between each 2 columns. Its value can be in the following form: tag{column-rule-width: nopx}

The column-rule-style property determines the shape of the borderline between each 2 columns. Its value can be one of the following:

tag{
  column-count : 2;
  column-rule-width : 2px;
  column-rule-style : normal
}

the borderline between each 2 columns will be in this shape

tag{
  column-count : 2;
  column-rule-width : 2px;
  column-rule-style : solid
}

the borderline between each 2 columns will be in this shape

tag{
  column-count : 2;
  column-rule-width : 2px;
  column-rule-style : dashed
}

the borderline between each 2 columns will be in this shape

tag{
  column-count : 2;
  column-rule-width : 2px;
  column-rule-style : dotted
}

the borderline between each 2 columns will be in this shape

tag{
  column-count : 2;
  column-rule-width : 2px;
  column-rule-style : double
}

the borderline between each 2 columns will be in this shape

The column-rule-color property is used to determine the color of the borderline between each 2 columns. Its value can be in one of the following forms:

form example
the name of the color red
the colour codes #cecf93
the redgreenblue function rgb(101, 150, 156)
the redgreenblueopacity function rgba(117, 138, 204, 0.68)

The column-rule property is used to collect the values of the column-rule-width, column-rule-style & column-rule-color properties together. it is witten in the following form:

selector{
  column-rule : column-rule-width column-rule-style column-rule-color
}
