The box-sizing property determines the start point of the tag dimensions. Its value can be one of the following:

value indication
border-box the borders & padding of the tag are considered part of the tag size
content-box the borders & padding of the tag are not considered part of the tag size

The width property represents the X dimension of the tag. Its value can be in one of the following form:

form indication
auto the width will be dependant on the content of the tag with its max value being the width of the outer tag
nopx the width will always be equal to that number independent upon its content
no% the width will always be equivelant to that percentage of the size of its outer tag independent upon its content

The min-width property determines a value the tag width can be above it depending on its content but can not be below it. The max-width property determines a value the tag width can be below it depending on its content but can not be above it.

The height property represents the Y dimension of the tag. Its value can be in one of the following form:

form indication
auto the height will be dependant on the content of the tag
nopx the height will always be equal to that number independent upon its content
no% the height will always be equivelant to that percentage of the size of its outer tag independent upon its content

The min-height property determines a value the tag height can be above it depending on its content but can not be below it. The max-height property determines a value the tag height can be below it depending on its content but can not be above it.

The margin property represents the space between this tag and the padding of the outer tag or any tag beside this tag. Its value can be in one of the following form:

form indication
auto the space will always be equal in both sides depending on the content of the tag
nopx the space in all sides will be equal to that number
nopx nopx the first no indicates the top & bottom space while the second no indicates the left & right space
nopx nopx nopx the first no indicates the top space while the second no indicates the left & right space and the third no indicates the bottom space
nopx nopx nopx nopx the first, second, third, fourth no indicates the top, right, bottom, left spaces by order
no% the space in all sides will always be equivelant to that percentage of the size of its outer tag
no% no% the first percentage indicates the top & bottom space while the second percentage indicates the left & right space
no% no% no% the first percentage indicates the top space while the second percentage indicates the left & right space and the third percentage indicates the bottom space
no% no% no% no% the first, second, third, fourth percentage indicates the top, right, bottom, left spaces by order

The margin property can be used with only one side of the selector or with each side alone forming the next four properties:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

The padding property represents the space between this tag and the margin of the its inner tags or text content. Its value can be in one of the following form:

form indication
nopx the space in all sides will be equal to that number
nopx nopx the first no indicates the top & bottom space while the second no indicates the left & right space
nopx nopx nopx the first no indicates the top space while the second no indicates the left & right space and the third no indicates the bottom space
nopx nopx nopx nopx the first, second, third, fourth no indicates the top, right, bottom, left spaces by order
no% the space in all sides will always be equivelant to that percentage of the size of this tag
no% no% the first percentage indicates the top & bottom space while the second percentage indicates the left & right space
no% no% no% the first percentage indicates the top space while the second percentage indicates the left & right space and the third percentage indicates the bottom space
no% no% no% no% the first, second, third, fourth percentage indicates the top, right, bottom, left spaces by order

The padding property can be used with only one side of the selector or with each side alone forming the next four properties:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

The resize property determines if the user can change the size of the tag. Its value can be in one of the following form:

tag{
  resize: none;
  border: 2px solid red
  overflow:hidden
}
the user can not resize it
tag{
  resize: horizontal;
  border: 2px solid red
  overflow:hidden
}
the user can resize it horizontally
tag{
  resize: vertical;
  border: 2px solid red
  overflow:hidden
}
the user can resize it vertically
tag{
  resize: both;
  border: 2px solid red
  overflow:hidden
}
the user can resize it in both directions

The z-index property determines which tag is in front of the others. Its value can be one of the following:

value indication
auto the tags will be ordered as they were written in the HTML file
number the tag with the greater number will be in front of the tag with the lower number
