The display property determines the status of the tag. Its value can be one of the following:

value indication
none as if the tag does not exist in the web page
inline the next tag is set beside it
block the next tag is set below it
inline-block the tag is inline but it is treated as block in certain properties
flex gives the inner tags a mixture of float & position in order to apply flex properties to them

The visibility property determines whether the tag is visible or not. Its value can be one of the following:

value indication
visible the user can see the tag and its contents
hidden the user can not see niether the tag not its contents but the tag exists in the web page and its place is kept clear of other tags

The backface-visibility property determines whether the back of the tag is visible or not. Its value can be one of the following:

tag{
  backface-visibility: visible;
  background-color: red
}
hover over me
tag{
  backface-visibility: hidden;
  background-color: red
}
hover over me

The overflow property determines whether the contents of the tag that are outside the space this tag holds are visible or not. Its value can be one of the following:

tag{
  overflow: visible;
  background-color: red
}
tag{
  overflow: hidden;
  background-color: red
}
tag{
  overflow: scroll;
  background-color: red
}
both scroll bars will always apper
tag{
  overflow: auto;
  background-color: red
}
scroll bar will apper in case of overflow

The overflow property can be used in only one dimention of the selector or in each dimention alone forming the next two properties:

  • overflow-x
  • overflow-y
