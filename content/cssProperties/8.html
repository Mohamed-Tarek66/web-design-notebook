The position property determines how to position the tag into the web page. Its value can be one of the following:

value indication
static the tag is positioned based on the html file and css style
fixed how ever the user moves this tag will be fixed on a single position on the screen independant upon the web page and his place based on the html file is kept free
relative the position of the inner tags are dependant on this tag
absolute styles it self by the css style based on the closest outside relative tag

Both the top & bottom properties determine the position of the tag into the web page by moving it vertically. Thier value can be in one of the following form:

tag{
  position: absolute;
  top: 20%;
  background-color: red
}
tag{
  position: absolute;
  top: 20px;
  background-color: red
}
tag{
  position: absolute;
  bottom: 20%;
  background-color: red
}
tag{
  position: absolute;
  bottom: 20px;
  background-color: red
}

Both the left & right properties determine the position of the tag into the web page by moving it horizontally. Thier value can be in one of the following form:

tag{
  position: absolute;
  left: 20%;
  background-color: red
}
tag{
  position: absolute;
  left: 20px;
  background-color: red
}
tag{
  position: absolute;
  right: 20%;
  background-color: red
}
tag{
  position: absolute;
  right: 20px;
  background-color: red
}

The perspective property represents the distance between the user and the tag. Its value can be in the following form: tag{perspective: nopx}

The transform-origin property determines the mass center of the tag in the X & Y directions. While the perspective-origin property determines the mass center of the tag in the Z direction. Thier value can be in one of the following form:

form indication
nopx nopx the first number represents the X coordinate & the second number represents the Y coordinate
no% no% the first percentage represents the X coordinate & the second percentage represents the Y coordinate
top left equivelant to 0% 0%
top center equivelant to 0% 50%
top right equivelant to 0% 100%
center left equivelant to 50% 0%
center center equivelant to 50% 50%
center right equivelant to 50% 100%
bottom left equivelant to 100% 0%
bottom center equivelant to 100% 50%
bottom right equivelant to 100% 100%

The float property determines if the tags are arranged beside each others or not. Its value can be one of the following:

value indication
none no float and the tags will be arranged as in the html file and css style
left the tags are arranged beside each others from left to right
right the tags are arranged beside each others from right to left

In order to clear the float one of the following methods are used:

  • giving the outer tag the following property tag{overflow: auto/hidden}
  • giving the tag following the floating tags the clear property. Its value can be one of the following:
value indication
left removes the float left properties
right removes the float right properties
both removes the float in all directions
