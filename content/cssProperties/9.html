The flex-direction property determines the direction at which items are arranged inside the flexbox. It is given to the flexbox tag. Its value can be one of the following:

value indication
row the items are arranged to the side of each others at the same direction given to the tag
row-reverse the items are arranged to the side of each others opposite to the direction given to the tag
column the items are arranged to below each others from top to bottom
column-reverse the items are arranged to above each others from bottom to top

The flex-wrap property determines if the items will continue to the next raw or continue outside of the flexbox. It is given to the flexbox tag. Its value can be one of the following:

value indication
nowrap the items will not continue to the next raw as it will continue outside of the flexbox
wrap the items will continue to the next raw from the top to the bottom
wrap-reverse the items will continue to the next raw from the bottom to the top

The flex-flow property is used to collect the values of the flex-direction & flex-wrap properties together. It is given to the flexbox tag. it is witten in the following form:

selector
{
  flex-flow : flex-direction flex-wrap
}

The order property determines the arrangment of the items inside the flexbox the lower order tag is arranged before the higher ones. It is given to the flex items. Its value can be in of the following form: tag{order: no}

The flex-grow property determines to what extent & at what ratio the flex items can expand inside the flexbox if there is space for expansion. It is given to the flex items. Its value can be in of the following form: tag{flex-grow: no}

The flex-shrink property determines to what extent & at what ratio the flex items can shrink inside the flexbox if the spae is tight. It is given to the flex items. Its value can be in of the following form: tag{flex-shrink: no}

The flex-basis property determines a static width for the flex items. It is given to the flex items. Its value can be in of the following form: tag{flex-shrink: nopx}

The flex property is used to collect the values of the flex-grow, flex-shrink & flex-basis properties together. It is given to the flexbox tag. it is witten in the following form:

selector
{
  flex : flex-grow flex-shrink flex-basis
}

The justify-content property determines how items are arranged horizontally inside the flexbox. It is given to the flexbox tag. Its value can be one of the following:

flexbox tag{
  display : flex;
  justify-content: flex-start
}
flexbox tag{
  display : flex;
  justify-content: flex-end
}
flexbox tag{
  display : flex;
  justify-content: center
}
flexbox tag{
  display : flex;
  justify-content: space-between
}
flexbox tag{
  display : flex;
  justify-content: space-around
}
