The cd command is used to get to a location on the current partition. The following table showes its forms:

form usage
cd / to bring you to the root of the partition you are on
cd partitionlitter:\folder1name\ to bring you to folder1 which is inside that partition
cd folder1name\ to bring you folder1 which is inside the current location you are on
cd folder1name\folder2name\ to bring you to folder2 which is inside folder1 which is inside the current location you are on
cd .. to bring you to the parent of the current folder with the location you are on
cd ..\folder1name\ to bring you folder1 which is beside the current folder with the location you are on

The partitionlitter: command is used to to bring you to the root of that partition.

The mkdir command is used to create a new folder. The following table showes its forms:

form usage
mkdir foldername to create a folder with that name inside the current folder with the location you are on
mkdir folder1name\folder2name to create folder2 with that name inside folder1 inside the current folder with the location you are on if folder1 does not exist it will creat it as well
mkdir location\foldername to create a folder with that name inside the folder with that location

The touch command is used to create a new file. The following table showes its forms:

form usage
touch filename to create a file with that name inside the current folder with the location you are on
touch foldername\filename to create file with that name inside that folder inside the current folder with the location you are on if the folder does not exist it will creat it as well
touch location\filename to create a file with that name inside the folder with that location

The dir command is used to provide information about the specified folder. The ls command is used to provide a list of the files & folders inside the specified folder.

The explorer command is used to open a file or folder in a new window. The following table showes its forms:

form usage
explorer to open mycomputer in new window
explorer .. to open the current folder with the location you are on in new window
mkdir location\ to open that folder with that location in new window
explorer location\filename to open that file which is inside the folder with that location in new window

The rename command is used to change the name of a folder or a file into a new name. It is written in the following form: rename file/foldername newname. The mv command is used to change the name of a folder or a file into a new name. It is written in the following form: mv file/foldername newname.

The mv command is also used to move a folder or a file to a new location. It is written in the following form: mv file/foldername location.

The cp command is used to creat a copy of a folder or a file in a new location. It is written in the following form:

cp filename location
cp -r foldername location

The rm command is used to remove a folder or a file from its current location. It is written in the following form:

rm filename
rm -r foldername

The grep command is used to search for files containing a certain body of text. The following table showes its forms:

form usage
grep "text" to search for files containing that text directly inside the current folder you are on
grep "text" location to search for files containing that text directly inside the folder with that location
grep "text" -r to search for files containing that text anywhere inside the current folder you are on
grep "text" -r location to search for files containing that text anywhere inside the folder with that location
