The git branch branchName command is used to creat a new branch with that name.

The git checkout branchName command is used to define that branch as the branch you are on.

The git checkout -b branchName command is used to creat a new branch with that name, then define it as the branch you are on.

The git branch command is used to determine all the branches in your current repo, thier names & the current branch you are on.

The git remote -v command is used to determine the name of the current repo you are on & the link of the remote repo.

The git branch -m branchNewName command is used to rename the current branch you are on.

The git branch -d branchName command is used to delete that branch if no changes are in it, while the git branch -D branchName command is used to delete that branch no matter what.

The git merge branchName command is used to merge that branch into the current branch you are on.
