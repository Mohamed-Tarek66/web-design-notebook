The git stash command is used to move all the files in the staging area to the stash, while the git stash save "message" command is used to move all the files in the staging area to the stash and link that message to them.

The git stash list command is used to form a list of the stashed files, their numbers & messages.

The git stash pop command is used to move the last stashed file from the stach back to the staging area, while the git stash pop stash@{no} command is used to move the stashed file with that number from the stach back to the staging area.

The git stash apply command is used to copy the last stashed file into the staging area, while the git stash apply stash@{no} command is used to copy the stashed file with that number into the staging area.

The git stash drop command is used to delete the last stashed file from the stach, while the git stash drop stash@{no} command is used to delete the stashed file with that number from the stach.

The git stash clear command is used to delete all stashed files from the stach.

The git stash show stash@{no} command is used to determine the files inside the stashed file with that number.
