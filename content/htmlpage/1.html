The HTML code must be constructed in the following form:

<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <meta name="text" content="text">
    <title> text </title>
    <style> </style>
    <link rel="stylesheet" href="css file location">
    <script> </script>
  </head>
  <body>
  </body>
</html>

<!doctype html> This tag indicates that this page is an HTML5 code.

<meta charset="utf-8"> This tag indicates the incoding of the website and It is usefull for accepting the arabic language.

<meta name=" " content="text"> This tag is used to form different types of meta that will be discussed later.

<title> text </title> This tag indicates the text that will appear in tab of the web browser.

<style> </style> In this tag we can edit the website and its different tags.

<link rel="stylesheet" href="css file location"> This tag is used to import stylesheet to edit the website from a css file.

<script> </script> This tag is used to link this page with a javascribt code.

</body> </body> This tag contains the codes forming the structure of the website.

<!-- text --> This tag indicates a comment in the HTML code and doesn't appear in the website
