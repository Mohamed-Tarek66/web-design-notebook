The img tag is a self-closed tag used to insert an image. The src attribute determines the location of the image: its value can be a link or a location on the pc. It is written in the following form: <img Src="the location of the image"/>

A common attribute used in conjuction with the img tag is alt. It is the text that will be written if the source of the imags is not found. This is shown in the following example.

<img Src="https://via.placeholder.com/50" alt="the source does not exist"/>
<img Src="" alt="the source does not exist"/>

"Placeholder" website provides empty images of different size. You have to put the link https://via.placeholder.com/no as the value of the src attribute (where no is the size of the images).
