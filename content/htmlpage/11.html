The audio tag is used to insert an audio file. It is written in the following form:

<audio>
  <source src="the location of the audio file" type="the mime type of the audio file">
  <source src="the location of the audio file" type="the mime type of the audio file">
  <source src="the location of the audio file" type="the mime type of the audio file">
  <source src="the location of the audio file" type="the mime type of the audio file">
  ..............
</audio>

Every source tag provides an audio file. In order to make sure that the web browsers provides the audio file with no problems we should put several source tags with different mime types. The src attribute determines the location of the audio file: its value can be a link or a location on the pc. The type attribute determines the mime type of the audio file which can be detemined through this link.

Common attributes used in conjuction with the audio tag are:

attribute usage
controls used to show the controls of the audio
autoplay indicates that the audio will autoplay after the loading of the website
loop used to make the audio automatically replay after it is finished
muted indicates that the audio will be muted by default
preload indicates when the audio file loads and it takes one of 3 values:
  • "auto" which indicates that the file will load with the website
  • "none" which indicates that the file will load only when played
  • "metadata" which indicates that the file will load only when played but its data will load with the website
