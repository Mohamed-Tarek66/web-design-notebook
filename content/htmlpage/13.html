The iframe tag is used to insert a window of a different website. It is written in the following form: <iframe src="the website link"> </iframe>. The src attribute determines the link of the website.

A common attribute used in conjuction with the iframe tag is scrolling. It determines whether ther is a scroll or not. it takes one of 3 values: "yes", "no" & "auto" which provides a scroll if the size of the iframe is lower than the size of the website. This is shown in the following example.

<iframe src="#" scrolling="yes"></iframe>
<iframe src="#" scrolling="no"></iframe>
<iframe src="#" scrolling="auto"></iframe>
