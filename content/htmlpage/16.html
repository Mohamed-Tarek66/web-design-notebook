The textarea tag is used to provide a textbox that can hold unlimited amount of text. It is written in the following form: <textarea> text </textarea>

Common attributes used in conjuction with the textarea tag are:

attribute usage
rows determines the vertical length of the textarea by determining the number of rows it shows
cols determines the horizontal length of the textarea by determining the number of litters that can be written in every raw
readonly the user can not change the value of the textarea
