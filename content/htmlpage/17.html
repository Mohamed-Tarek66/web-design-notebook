</select>

The option tag is used to provide one choices in the selectbox. The optgroup tag is used to gather every group of options together.

Common attributes used in conjuction with the select tag are:

attribute usage
disabled the user can not choose an option from the select tag
size determines the vertical length of the select by determining the number of options it shows
multiple allows the user to choose multiple options at the same time

Common attributes used in conjuction with the option tag are:

attribute usage
value represents the value of the option sent to the programming file in the form "name of the select = value of the option"
selected indicates that this option will be selected by default

An attributes used in conjuction with the optgroup tag is label which represents the text that will be written in the select to define that group
