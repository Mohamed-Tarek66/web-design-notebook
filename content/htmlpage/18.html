The progress tag is used to provide a bar filled in a certain percent. It is used to describe a procees and how much it is done. It is written in the following form: <progress value="number" max="number"> tags </progress>

Attributes used in conjuction with the progress tag are:

attribute usage
max determines the value that fills the bar to 100%
value determines percentage filled of the bar as it equals (value/max)

The meter tag is used to provide a bar filled in a certain percent. It is used to determine the value of a certain parameter that can increase or decrease with time. It is written in the following form: <meter value="number" max="number" min="number"> tags </meter>

Attributes used in conjuction with the meter tag are:

attribute usage
max determines the value that fills the bar to 100%
max determines the value the bar is empty at (0%)
value determines percentage filled of the bar as it equals ((value-min)/(max-min))
low determines the value that the bar defines it as a low value((low-min)/(max-min))
high determines the value that the bar defines it as a high value((high-min)/(max-min))
optimum determines the value that the bar defines it as a optimum value((optimum-min)/(max-min))
