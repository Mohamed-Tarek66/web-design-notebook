The div tag is the building unit of the body. It is used to separate the body into different parts and is very handy when editing. It is written in the following form: <div> tags </div>

Common tags used to replace the div tag for different features are:

tag usage
section used to separate the body into different sections and It is written in the following form: <section> tags </section>
article used to separate the body into defferent articles and It is written in the following form: <article> tags </article>
span used to separate a part of a tag in order to edit it separately and It is written in the following form: <span> tags </span>
header contains the tags that will appear at the top of the website usually the title of the website and the navigation bar and It is written in the following form: <header> tags </header>
footer contains the tags that will appear at the bottom of the website usually the contacts and the social media links and It is written in the following form: <footer> tags </footer>
main indicates that this tag is the main div of the website and It is written in the following form: <main> tags </main>
nav used to form the navigation bar, It usually contains an unordered list and each list item will represent some of the contents of the website and It is written in the following form: <nav> tags </nav>
aside used to form the side bar and It is written in the following form: <aside> tags </aside>
form used to group several input tags and the caption of every image and It is written in the following form: <form> tags </form>
figure used to group several images and It is written in the following form:
<figure>
  <img Src="the source of the image" alt="text"/>
  <figcaption> the discribtion of the image</figcaption>
  <img Src="the source of the image" alt="text"/>
  <figcaption> the discribtion of the image</figcaption>
</figure>
The figcaption tag provides a discribtion for every image and it can also contain other tags
time used to provide a div with associated with certain date and time and It is written in the following form: <time datetime="a certain date and time">tags</time>
The datetime attribute determines the date and time associated with the time tag and its value is written if the following form: "year-month-day hr:minute:second"
dialog used to provide a div that will move with the user along the web page and It is written in the following form: <dialog open>tags</dialog>
The open attribute indicates that the dialog will be visible for the user

Attributes used in conjuction with the form tag are:

attribute usage
name determines the name that the programming language will recognize the form with
action determines the location of the programming file that will recieve the input values of this form
method determines the way the programming file that will recieve the input values of this form it accepts 2 values:
  • "post" which will send the data to the programming file and post it besides the widsite address
  • "get" which will only send the data to the programming file
novalidate removes the restriction of the values of the inputs inside this form
target It holds 2 values:
  • "_self" which makes the values of the inputes is submitted in the same web tab
  • "_blank" which makes the values of the inputes is submitted in a new web tab
enctype determines the encoding of the data sent to the programming file and can only be used if the method is post and it accepts one of the following values:
  • "application/x-www-form-urlencoded"
  • "multipart/form-data"
  • "text/plain"
