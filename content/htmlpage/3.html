The paragraph tag is used to hold text in it. It is written in the following form: <p> tags </p>

The blockquote tag is used to hold text in it but its first line is after some space. It is written in the following form: <blockquote> tags </blockquote>

The pre tag is used to hold text in it with all the spaces in the text are innigligable. It is written in the following form: <pre> tags </pre>

The code tag is used to show text in code font. It is written in the following form: <code> tags </code>

The label tag is used to hold text in it usually used for discribtion. It is written in the following form: <label> tags </label>

The output tag is used to hold a result of programming process in it. It is written in the following form: <output> tags </output>
