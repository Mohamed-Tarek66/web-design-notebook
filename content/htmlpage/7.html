The table tag is used to align data in form of a table. It is written in the following form:

<table>
  <thead>
    <tr>
      <td> cell content </td>
      <td> cell content </td>
      <td> cell content </td>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td> cell content </td>
      <td> cell content </td>
      <td> cell content </td>
    </tr>
    <tr>
      <td> cell content </td>
      <td> cell content </td>
      <td> cell content </td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td> cell content </td>
      <td> cell content </td>
      <td> cell content </td>
    </tr>
  </tfoot>
</table>

Tags used in conuction with the tabble tag are:

tag usage
caption represents the title of the table
thead represents the first few rows of the table
tfoot represents the last few rows of the table
tbody represents the raws between the head and the foot of the table
tr represents one raw of the table
td represents one cell of the table
th represents one aligned-center cell of the table written in bold text

A common attribute used in conjuction with the table tag is border which determines the size of the borders of the table

attribute usage
