Common tags used to apply different features to text are:

tag usage
break a self-closed tag used to end the line and start from the next line and it is written in the following form: text <br/> text
wordbreak used to break the text into multiple lines and it is written in the following form: text <wbr> text </wbr> text
small used to provid a text smaller than the default font size and it is written in the following form: <small> text </small>
strong used to provid a text bigger than the default font size and it is written in the following form: <strong> text </strong>
bold used to provid a bold text and it is written in the following form: <b> text </b>
italic used to provid an italic text and it is written in the following form: <i> text </i>
em used to provid an italic text that is so important that it may change the meaning of the sentance if removed and it is written in the following form: <small> text </small>
delete used to provid a text with solid line over it to indicate that the text is removed and it is written in the following form: <del> text </del>
ins used to provid a text with solid line under it to indicate the correction of other text and it is written in the following form: <ins> text </ins>
mark used to provid a highlighted text and it is written in the following form: <mark> text </mark>
quote used to provid a quoted text and it is written in the following form: <q> text </q>
abbr used to provid an explaination for text that appears when the user hover over it and it is written in the following form: <abbr title="explaination"> text </abbr>
sup used to provid a text as an exponential and it is written in the following form: <sup> text </sup>
sub used to provid a text as a small down text and it is written in the following form: <sub> text </sub>
horizontal line used to provid a horizontal solid line between tags and it is written in the following form: tag <sub/> tag
bdi used to avoid problems when writing arabic text within english text and it is written in the following form: english text <bdi> arabic text </bdi> english text
