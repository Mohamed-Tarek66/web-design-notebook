The hyperlink tag is used to provide a text that monves you on click to its refference. It is written in the following form: <a> text </a>

In order to determine a refference to the hyperlink we use the attribute href. The refference could be inside or outside this web page. We can determine the refference as this page by using "#"" as the value of the href attribute. We can determine the refference as a tag in this page by using "#the class of the tag" as the value of the href attribute. We can determine the refference as an e-mail to aspecific mail by using "mailto:.....@...." as the value of the href attribute. This is shown in the following example.

<a href="https://www.w3schools.com/"> click me to visit w3schools</a>
<a href="#"> click me to go to the same page</a>
<a href="#the ID of the tag"> click me to go to the construction form of the HTML code</a>
click me to visit w3schools

A common attribute used in conjuction with the hyperlink tag is target. It holds 2 values:

  • "_self" which makes the hyperlink open its refference in the same web tab
  • "_blank" which makes the hyperlink open its refference in a new web tab

This is shown in the following example

<a href="https://www.w3schools.com/" target="_self"> click me to visit w3schools</a>
<a href="https://www.w3schools.com/" target="_blank"> click me to visit w3schools</a>
